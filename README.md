Electronics sales data project/ Power BI project #1
# Sales Dashboard - Power BI

Multi-currency sales analysis with profit margins and USD conversion.

## Features
- All sales converted to USD (CAD rate: 1 USD = 0.76 CAD)
- Profit Margin % calculations
- Time intelligence (Year, Quarter, Month, Day)
- Interactive slicers by Region, Product, Date

## How to use
1. Download the `.pbix` file
2. Open with Power BI Desktop
3. Refresh the data if needed

## Built with
- FactSales table
- Date dimension
- ExchangeRates table
