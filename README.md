Electronics sales data project/ Power BI project #1

# Sales Dashboard - Power BI

Multi-currency sales analysis with profit margins.

## Features
- Profit Margin % calculations
- Time intelligence (Year, Quarter, Month, Day)
- Interactive slicers by Product Category, Product, Date

## How to use
1. Download the `.pbix` file
2. Open with Power BI Desktop
3. Refresh the data if needed

## Built with
- FactSales table
- Date dimension
